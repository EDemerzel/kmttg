Web server pages

This web server is still a work in progress, offering a small subset of kmttg capabilities via a web browser, and also adding streaming video to browser capabilities. The Video Streaming capability only works for browsers supporting HLS and has only been tested on iOS Safari, but should work for Android Chrome as well.
These pages should be accessed via kmttg web server only, not directly as a file with a web browser. For documentation on enabling kmttg web server consult the kmttg web interface Wiki page.

Choose which web server page you want here once connected to kmttg web server:
  • Video Streaming
  • Share Browser
  • Remote
  • ToDo
  • Season Passes
  • Info
  • Job Monitor
